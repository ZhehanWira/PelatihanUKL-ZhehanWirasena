# Presensi App

## Deskripsi
Presensi App adalah aplikasi untuk mencatat kehadiran siswa atau pengguna secara online.  
Aplikasi ini memudahkan guru atau admin untuk melihat rekap kehadiran harian maupun bulanan, menambahkan data kehadiran, dan mengelola data pengguna.  
Program ini dibuat menggunakan **Next.js** dengan **Tailwind CSS** untuk tampilan dan **Recharts** untuk grafik rekap kehadiran.

## Fungsi dan Cara Kerja
1. **Login**
   - Pengguna memasukkan username dan password untuk masuk ke aplikasi.
   - Jika login sukses, pengguna diarahkan ke dashboard.
2. **Dashboard**
   - Menampilkan ringkasan data kehadiran.
   - Menampilkan tombol navigasi ke halaman Presensi, Rekap Kehadiran, dan Data Pengguna.
3. **Presensi**
   - Admin bisa menambahkan data kehadiran harian pengguna.
   - Status kehadiran: Hadir, Izin, Alpa.
4. **Rekap Kehadiran**
   - Menampilkan grafik kehadiran bulanan menggunakan Recharts.
   - Menampilkan ringkasan total hari hadir, izin, dan alpa.
5. **Data Pengguna**
   - Menampilkan daftar pengguna.
   - Admin dapat menambah, mengedit, atau menghapus data pengguna.
